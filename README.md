# VSpider custom firmware

**Yet another firmware for VSpider project**

Config switches:

- SW1 - switch between Gluk (ZC) and EsxDOS (DivMMC). Default is DivMMC
- SW3 - DiagROM on/off. Default is off

ROM layout:

- ROM A16=0: Gluk + trdos + basic128 + basic48
- ROM A16=1: Esxdos + Diagrom + basic128 + basic48
